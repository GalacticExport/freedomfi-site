How to Spot Scams (Quick Hits)

  • Any DM with a “CA” before launch is fake.
  • Seed phrase requests = instant scam.
  • Fake “support” accounts mimic usernames.
  • Links with weird domains or extra characters are traps.
  • Pressure tactics: “act now” = red flag.
Rule: Verify links only from this site + pinned Telegram.

Wallet Security Basics

  • Use a hardware wallet for long-term holdings.
  • Create a “burner wallet” for new apps.
  • Never sign random transactions you don’t understand.
  • Bookmark official sites. Don’t click paid ads.
  • Revoke approvals periodically (where supported).
Habit: Double-check addresses before sending.
